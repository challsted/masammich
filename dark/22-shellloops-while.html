While Loops!

Types of loops

There are several types of loops, and each type has a different purpose, we will be covering the following types of loops.
  • if
  • while
  • for

What is a "loop"?

They are called this because they "loop" or iterate though data or instructions until they are no longer true. To be able to understand what I mean by "until they are no longer true", you must understand what is known as "Boolean Logic". Boolean logic (very simplified) is a "true or false" "1 or 0", basically, its always one way or the other.

What is an "while" loop?

A while loop is a little like a if loop. As in they check if something is true right off the bat, where they start to differ is if they are true, they continue doing what you tell them to, until they are false. This opens up the chance of an infinite loop (usually considered a bad thing).

A while loop only has 5 things it has to have, and several optional things:

  • A while loop Must have:
    1. They always opens with the word "while"
    2. A condition that must be true
    3. On the next line the word "do" or on the same line with a semicolon after "do;"
    4. A command to do while the condition is ture
    5. The word "done" when the while loop is completed (with 1 iteration)
  • A while loop can include:
    1. A way to end the loop
    2. ???

An example while loop:
FIVE=5
while [ $FIVE = 5 ]
do
  echo "$FIVE is equal to 5!"
done

You might be able to see, even with out running this program, that this will cause what is called an "infinite loop". This loop will never be able to stop running because 5 will always be equal to 5!
But this is a perfectly functioning example of a while loop. In this case you will have to type hit CTRL+C on your keyboard to "Cancel" the script.
Lets try to write a script that will stop eventually!

A=1
while [ $A -lt 10 ]
do
  echo "A is currently $A"
  A=`expr $A + 1`1
done

1 expr is to "evaluate expressions (man expr)
This will output the following:
A is currently 1
A is currently 2
A is currently 3
A is currently 4
A is currently 5
A is currently 6
A is currently 7
A is currently 8
A is currently 9

This is exactly what we want it to do! It printed the value of A until it no longer was true.
We gave our script the varible "A", A was holding the value "1".
We then told the script "while the value of A is less then 10 tell me that A is currently a value, then add 1 to the value of A"
As you can see it did not print that A was equal to 10, because of the way we worded our condition. "$A -lt 10" or "A is less then 10". So, it stopped doing anything when $A was equal to 10. We could of said "$A -le 10" and it would have printed "A is currently 10" or we could of said "$A -le 11" and it would of stopped at 10.

Shell Embedded Loops

Something cool you can do with loops, is you can put a loop INSIDE a loop!
So, for example:
A=1
if [ $A -lt 101 ]
then
  echo "A is equal to $A"
  while [ $A -lt 101 ]
  do
    A=`expr $A + 1`
  done
fi

As you will see, I set a varible of "A" to 1, then I open a if loop and I check if the value of A is less then 101 (which at the moment it is), then as the command for the if loop, I execute another loop (in this case a while loop) which also checks if the value A is less then 101. The command for the while loop is then to echo the current value of A, then it preforms a little math on the value of A and assigns it to A, then ends both loops.
This simply prints "A is equal to N" 100 times.
