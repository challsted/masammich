For Loops!

Types of loops

There are several types of loops, and each type has a different purpose, we will be covering the following types of loops.
  • if
  • while
  • for
  • until

What is a "loop"?

They are called this because they "loop" or iterate though data or instructions until they are no longer true. To be able to understand what I mean by "until they are no longer true", you must understand what is known as "Boolean Logic". Boolean logic (very simplified) is a "true or false" "1 or 0", basically, its always one way or the other.

What is an "for" loop?

A for loop is a little like a while loop. As in they repeate an action over and over and over. Where they differ is that a for loop preforms an action a specific number of times. This number can be set statically, or it can be pulled in from a variable that can change. Once it is complete, it will stop. This still has a chance of creating an infinite loop, so dont think your out of the water yet!

A for loop has several needed pieces:

  1. They always open with the word "for"
  2. A variable, or static command that needs to be true
    • This can be changed with the next part though
  3. (Optional, but almost always used) the word "in"
  4. Another Variable or static command that the previous needs to be apart of
  5. The word "do"
  6. Something to happen for each of the above
  7. The word "done"
I have read it somewhere like this: "for (Each Item) in (A list of Items) do (Command(s))"
If that sentence helps you (I know it helped me) feel free to use it/remember it!

Here is an example for loop:
for i in 1 2 3 4 5
do
  echo "The Variable i is currently $i"
  j=`expr $i \* 10` (Make sure you escape the * or it wont work!)
  echo "$j is $i times 10!"
done
This will output the following:
The Variable i is currently 1
10 is 1 times 10!
The Variable i is currently 2
20 is 2 times 10!
The Variable i is currently 3
30 is 3 times 10!
The Variable i is currently 4
40 is 4 times 10!
The Variable i is currently 5
50 is 5 times 10!

Kind of neat huh? Its like a controlled while loop.
Lets change the above loop just a little to show you what else it can do
A=5
for i in $A
do
  echo "The Variable i is currently $i"
  j=`expr $i \* 10`
  echo "$j is $i times 10!"
done
This will produce the exact same results, its 1 line longer then the origional, but it turns the "1 2 3 4 5" into a variable for the example factor of it.

There are many ways to write the opening line of a for loop

I will attempt to list as many of them as I can think of, and as I can find via simple google searches. I will explain them also. I will only list additonal details if needed, so i will only be listing the first "line" of the loop for example purposes.
Please note these DO NOT work in SH, but they do in more advanced shells like Bash.

C/C++ Like

Explained:
for (( expr1; expr2; expr3 ))
Example:
for (( i=1; i <= 10; i++ ))
For those that dont know i++ simply increments i by 1
This will print Number: 1 all the way to Number: 10

Range of Numbers

Explained:
for num in {START..END}
Example:
for num in {1..20}
do
  echo "Number: $num"
done
In this case the ".." is a kind of "filler" for everything between the numbers on each side.
This will print Number: 1 all the way to Number: 20

Incremental Range of Numbers

Explained:
for num in {START..END..COUNT_BY}
Example:
for num in {1..99..3}
do
  echo "Number: $num"
done
Just like above, the first ".." is a filler, but the 2nd ".." is saying "count by this".
This will count from 1 to 99, by 3's and print it like Number: 67, do you know why its 67, and not 66?
